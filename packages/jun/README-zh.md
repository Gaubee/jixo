# @jixo/jun

`@jixo/jun` 是一个有状态的 Shell 代理工具，旨在管理和持久化 Shell 命令的执行过程。它允许您无缝地运行命令、跟踪历史记录、查看输出并管理后台进程。

## 特性

- **有状态执行**: 为每个运行的命令持久化元数据和 stdio。
- **后台任务**: 轻松启动和管理长时间运行的后台进程。
- **历史跟踪**: 查看所有已执行命令的历史记录。
- **日志审查**: 检索并审查任何任务的完整 stdio 日志。
- **进程管理**: 列出并终止正在运行的后台任务。
- **本地与全局上下文**: 在本地 `.jun` 目录中操作，或回退到全局的 `~/.jun` 目录。
